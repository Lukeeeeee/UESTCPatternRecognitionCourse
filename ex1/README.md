#### 如何运行代码
Prerequisite：
```shell
1. Octave >=4.0 (没有使用Matlab)
2. Octave image package （已经放在了压缩包里，运行 main 程序会自动安装）
```
Run:
1. 打开 Octave
2. 改变当前路径到该作业文件夹
3. 在Octave Command 中输入：
	```shell
		main
	```
	按照提示运行即可，共有三个算法实现，具体代码分别在 tyc，tyc2和 EM 文件中，可单独查看，也可分别单独运行。
